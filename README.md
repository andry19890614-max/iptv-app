# 📡 IPTV Player — Android додаток

M3U плеєр для перегляду IPTV каналів на Android.

## Що вміє додаток
- Завантаження M3U/M3U8 плейлістів (за URL або вставкою тексту)
- Автоматичне групування каналів по категоріях
- Відтворення HLS потоків (живе ТВ)
- Пошук каналів
- Обране (зірочки)
- Два режими відображення: сітка та список
- Збереження плейлістів між запусками

## Покрокова інструкція збірки APK

### Крок 1: Встанови Node.js
1. Зайди на https://nodejs.org
2. Натисни зелену кнопку **LTS**
3. Встанови як звичайну програму

### Крок 2: Встанови Android Studio
1. Зайди на https://developer.android.com/studio
2. Завантаж і встанови
3. При першому запуску вибери **Standard Setup**

### Крок 3: Розпакуй цю папку
Розпакуй ZIP у будь-яке місце на комп'ютері.

### Крок 4: Відкрий термінал
- **Windows**: Win+R → cmd → Enter
- **Mac**: відкрий Terminal

### Крок 5: Перейди в папку проєкту
```
cd шлях/до/iptv-app
```

### Крок 6: Виконай ці команди по черзі

```bash
# 1. Встанови залежності (1-3 хвилини)
npm install

# 2. Збери проєкт
npm run build

# 3. Ініціалізуй Capacitor
npx cap init IPTVPlayer com.myapp.iptv --web-dir=build

# 4. Додай Android
npx cap add android

# 5. Синхронізуй
npx cap sync

# 6. Відкрий в Android Studio
npx cap open android
```

### Крок 7: Збери APK в Android Studio
1. Зачекай поки завершиться індексація (прогрес внизу)
2. Меню: **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**
3. Натисни **locate** в повідомленні внизу
4. Файл `app-debug.apk` — твій готовий додаток!

### Крок 8: Встанови на телефон
- Перекинь APK на телефон (USB, Telegram, Google Drive)
- Відкрий файл → дозволь встановлення → Встановити

## Як користуватись

1. Відкрий додаток
2. Натисни "Завантажити плейліст"
3. Вставь URL на M3U файл (або скопіюй текст плейліста)
4. Натисни "Завантажити"
5. Вибери канал і дивись!

## Де взяти M3U плейлісти?
Пошукай в Google: "IPTV m3u playlist free" або "iptv ukraine m3u".
Зазвичай це посилання виду: `https://example.com/playlist.m3u`

## Якщо щось не працює

**npm не знайдено** → Перезавантаж комп'ютер після встановлення Node.js

**Помилка при build** → Видали папку node_modules і запусти `npm install` знову

**Канал не відтворюється** → Деякі потоки можуть бути недоступні або заблоковані
